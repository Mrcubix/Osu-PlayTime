Ce site fais partie d'une projet exterieur de ma création consistant à enregistrer les durées de sessions pour le jeu osu!

Dans l'onglet PlayTime, vous trouverez la durée de vos sessions sur les beatmapset jouées.

Un beatmapset est un ensemble de maps publié par un joueur mais pouvant être créer par tous le monde, Quant aux maps, il s'agit d'un ensemble de patternes suivant ou non la bande sonore associé.

Les données sont collectés à l'aide d'un programme open source lisant la mémoire tandis que je collecte les informations délivrées par cette dernière.

Ces informations sont pour le moment stocké pour chaque joueurs dans un fichier json. Le javascript associé à chaque page recherche des informations soit dans les données du joueurs, Soit de tous les joueurs utilisant l'outil (WIP).

Page PlayTime:

Exemple de container:

le temps total joué sur chaque beatmap est représenté par le texte "Total time played: + Temps total". Si vous cliquez sur un beatmapset, vous accédez à son contenu c'est à dire les différentes niveaux de difficultés (aussi appelé maps). à chaque difficultés sont associés les différents temps passé sur chaque niveaux de difficultés, représenté par le texte "Time played: + Temps".

En arrière plan, le code javascript demande le contenu du json pour le joueur et ajoute les beatmapsets sous différent containers à la page. ces containers se voient attribué un eventlistener qui vérifie si le container est cliqué, si il est cliqué, alors le script recherche les difficultés de la map et les ajoute chacun dans un sous-sous-container du beatmapset.

beatmapset ☛ nom + temps total + container de toutes difficultés ☛ container difficulté ☛ nom + temps
